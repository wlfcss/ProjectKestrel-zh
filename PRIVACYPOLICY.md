# 隐私政策

**生效日期：2026 年 3 月 21 日**

翎鉴（Project Kestrel 中文版）尊重您的隐私。

## 数据收集

翎鉴**不收集任何数据**。所有照片处理均在您的本地设备上完成，软件不会向任何外部服务器发送信息。

具体而言：
- 不收集设备标识符或机器信息
- 不收集使用统计或分析数据
- 不收集文件名、路径或图片内容
- 不收集个人身份信息
- 软件运行无需网络连接

## 与上游版本的区别

原版 Project Kestrel 包含匿名遥测功能（安装统计、分析完成计数等）。本分支已完全移除所有遥测代码。

## 联系

如对本隐私政策有疑问，请通过 GitHub Issues 联系。
