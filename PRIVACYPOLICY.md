# Privacy Policy

**Effective Date: March 1, 2026**

Project Kestrel respects your privacy. This policy explains what data we collect and why.

## 1. Data Collection

Project Kestrel is designed to work entirely offline. However, it collects minimal telemetry to help us understand usage and improve the software.

### A. Non-Optional Telemetry
The following data is collected automatically and cannot be disabled:
- **Machine Identifier:** A randomly generated UUID that is stable on your system. This is NOT linked to your hardware serial numbers, username, or IP address.
- **Platform Info:** Your Operating System type (e.g., Windows, macOS) and version.
- **Usage Volume:** When an analysis folder is completed, we receive the count of images successfully analyzed.

### B. Optional Analytics (Opt-In)
If you choose to opt-in, we may collect:
- Detailed performance stats (analysis speed).
- File formats used (e.g., CR3, JPG).

## 2. What We NEVER Collect
- **NO Filenames or Folder Paths.**
- **NO Image Content or Thumbnails.**
- **NO Personally Identifiable Information (PII)** like names, emails (unless provided in a feedback form), or location data.

## 3. Data Usage
This data is used solely to track the software's adoption and to prioritize performance improvements for the most common platforms and file formats. We do not sell or share this data with third parties.

## 4. Contact
If you have questions about this policy, please use the "Feedback" button within the application.
