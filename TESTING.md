# 翎鉴 — UI/UX 测试计划

> **使用方法:** 按顺序执行每一步。标记 `[x]` 通过，`[!]` 有问题，`[-]` 不适用。

---

## 核心测试

---

### Step 0 — 启动

- [ ] 启动应用，主界面正常加载（顶栏 + 空状态页面）
- [ ] 空状态显示 Logo、提示文字和"打开文件夹"/"分析新文件夹"按钮
- [ ] 顶栏包含：排序下拉框、分组下拉框、导入/分析/设置/导出按钮

---

### Step 1 — 分析文件夹

点击"分析"按钮。

- [ ] 弹出分析文件夹对话框，提示选择根目录
- [ ] 文件夹树正确显示子文件夹
- [ ] 已分析的文件夹有视觉区分（颜色标签）
- [ ] 勾选文件夹后可开始分析
- [ ] 进度条实时更新
- [ ] 可暂停/恢复/取消分析
- [ ] 分析完成后自动加载结果到场景网格

---

### Step 2 — 浏览场景

点击"导入"打开已分析的文件夹。

- [ ] 场景卡片正确显示：缩略图、场景编号、物种名、画质分数
- [ ] 缩略图正常加载（懒加载）
- [ ] 状态栏显示场景数和照片数
- [ ] 文件夹分组头显示文件夹名和场景数
- [ ] 分组头可折叠/展开
- [ ] 分组头有"关闭"、"打开"、"重置筛选决定"按钮
- [ ] 点击"关闭"按钮回到空状态

---

### Step 2.1 — 场景详情

点击场景卡片打开详情。

- [ ] 详情面板显示场景内所有图片
- [ ] 图片显示文件名、物种、画质分数、星级
- [ ] 可通过键盘快捷键在图片间导航
- [ ] 双击图片在配置的编辑器中打开
- [ ] Escape 关闭详情，场景网格数据保留

---

### Step 3 — 评分与保存

- [ ] 点击星级可设置评分（1-5）
- [ ] 手动评分与 AI 评分视觉区分
- [ ] "保存"按钮写入 CSV
- [ ] "还原"按钮恢复到上次保存状态
- [ ] 合并场景功能正常

---

### Step 4 — 设置

- [ ] 设置对话框正常打开
- [ ] 设置项正确回显已保存的值
- [ ] 修改后的设置在重启后保持

---

### Step 5 — 导入/取消/切换

- [ ] 打开文件夹 A → 场景正常显示
- [ ] 打开文件夹 B → 正确切换到 B 的数据
- [ ] 打开文件夹 → 取消 → 当前数据保留
- [ ] 关闭文件夹 → 回到空状态 → 重新打开 → 数据正常

---

### Step 6 — 元数据写入

- [ ] 文件夹分组头的"写入照片元数据"按钮可用
- [ ] XMP sidecar 文件正确生成
- [ ] 元数据包含正确的星级和物种信息

---

### Step 7 — 重启一致性

- [ ] 关闭应用后重新打开
- [ ] 打开相同文件夹，所有数据完整保留
- [ ] 设置值不变

---

## 细节测试

---

### 排序与筛选

- [ ] 排序下拉框切换（按画质/按时间等）卡片正确重排
- [ ] 分组模式切换正常
- [ ] 多选（Ctrl+Click / Shift+Click）正常
- [ ] Escape 清除多选

---

### 分析队列

- [ ] 多文件夹入队分析
- [ ] 队列进度面板正常显示
- [ ] 暂停/恢复/取消行为正确
- [ ] 分析完成后自动加载对应文件夹的数据

---

### 边界情况

- [ ] 打开无 `.lingjian` 数据的文件夹 → 显示友好错误提示，不崩溃
- [ ] 空 CSV 或损坏的 CSV → 显示错误，不崩溃
- [ ] 连续快速切换文件夹 → 最终状态正确，无竞态错误
- [ ] 3 次以上打开/关闭循环 → 无内存/定时器泄漏
